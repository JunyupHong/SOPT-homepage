# SOPT 홈페이지


## HISTORY 업데이트
### history 페이지 업데이트
  - public/data/history/index.json 에 해당 기수 정보 입력
    - history 페이지의 임원진 버튼 생성
  
### history/{기수} 페이지 업데이트
  - public/data/{기수}.json 파일 생성 및 정보 입력
    - history/{기수} 페이지에 데이터 반영
    - 이미지 URL은 /assets/history/{기수}/{파일명} 형식
  - 위 파일에 입력한 이미지 URL에 해당하는 경로에 이미지 업로드
